# Package index

## System Setup

Create a simulation and add bodies

- [`create_system()`](https://daverosenman.github.io/orbitr/reference/create_system.md)
  : Initialize an orbitr simulation system
- [`add_body()`](https://daverosenman.github.io/orbitr/reference/add_body.md)
  : Add a physical body to the system

## Simulation

Run the N-body simulation

- [`simulate()`](https://daverosenman.github.io/orbitr/reference/simulate.md)
  : Simulate kinematics for an orbitr system

## Analysis & Visualization

Transform and plot results

- [`shift_reference_frame()`](https://daverosenman.github.io/orbitr/reference/shift_reference_frame.md)
  : Shift the coordinate reference frame of the simulation
- [`plot_orbits()`](https://daverosenman.github.io/orbitr/reference/plot_orbits.md)
  : Plot Orbital Trajectories (Smart 2D/3D Dispatch)
- [`plot_orbits_3d()`](https://daverosenman.github.io/orbitr/reference/plot_orbits_3d.md)
  : Plot 3D Interactive Orbital Trajectories
